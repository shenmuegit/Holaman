# Holaman API 调试工具功能列表

Holaman 是一个 API 调试工具。当前规划将核心调试、接口测试、Mock、文档、WebSocket、GraphQL、版本管理等能力统一纳入第一阶段，优先建设成一个完整可用的接口研发工作台。

## 第一阶段功能

### 1. 请求构建

- 支持 `GET`、`POST`、`PUT`、`PATCH`、`DELETE`、`HEAD`、`OPTIONS` 等 HTTP 方法。
- 支持 URL 输入、Query 参数编辑和参数启用/禁用。
- 支持 Headers 编辑、批量编辑和常用 Header 自动补全。
- 支持 Body 编辑：
  - `JSON`
  - `form-data`
  - `x-www-form-urlencoded`
  - raw text
  - XML
  - binary
- 支持 Cookie 管理。
- 支持请求超时设置。
- 支持 SSL 证书校验开关。
- 支持代理配置。

### 2. 响应查看

- 展示状态码、耗时、响应大小。
- 展示 Response headers。
- 展示 Cookies。
- 支持 Body 格式化查看。
- 支持 JSON 高亮、折叠、搜索。
- 支持原始响应查看。
- 支持文本、HTML、图片、XML 等基础预览。
- 支持响应内容复制和保存。

### 3. 环境变量

- 支持多环境，例如开发、测试、预发布、生产。
- 支持变量引用，例如 `{{base_url}}`。
- 支持全局变量、环境变量、集合变量和请求级临时变量。
- 支持变量启用/禁用。
- 支持敏感变量脱敏展示。
- 支持从响应中提取字段并写入变量。

### 4. 请求集合

- 支持创建 Collection。
- 支持文件夹分组。
- 支持保存请求。
- 支持复制、重命名、删除请求。
- 支持请求排序。
- 支持最近请求历史。
- 支持 Collection 内搜索。

### 5. 鉴权支持

- Bearer Token。
- Basic Auth。
- API Key。
- OAuth 2.0 基础支持。
- Digest Auth。
- 自定义 Header 鉴权。
- 支持集合级、文件夹级、请求级鉴权配置。

### 6. 请求前后脚本

- 支持 Pre-request script。
- 支持 Tests / post-response script。
- 支持从响应中提取 token 或字段写入变量。
- 支持简单断言：
  - 状态码
  - 响应字段
  - 响应时间
  - Header 是否存在
  - Body 是否包含指定内容
- 支持脚本执行日志。

### 7. 文件上传与 Multipart

- 支持多文件上传。
- 支持 `form-data` 字段类型区分：文本 / 文件。
- 支持上传进度展示。
- 支持上传错误提示。
- 支持文件字段启用/禁用。

### 8. 请求历史与回放

- 自动保存请求历史。
- 支持按时间、状态码、URL、方法搜索历史。
- 支持从历史恢复请求。
- 支持重新发送历史请求。
- 支持对比两次请求或响应差异。
- 支持清理历史记录。

### 9. 接口测试

- 支持 Collection Runner。
- 支持批量运行请求。
- 支持测试报告。
- 支持失败请求定位。
- 支持环境切换运行。
- 支持请求执行顺序控制。
- 支持失败后继续执行或中断执行。

### 10. 数据驱动测试

- 支持 CSV / JSON 数据源。
- 支持参数化请求。
- 支持多组数据循环执行。
- 支持每组数据测试结果记录。
- 支持数据源字段自动映射为变量。

### 11. Mock 服务

- 支持根据接口定义创建 Mock。
- 支持自定义 Mock 响应。
- 支持延迟响应。
- 支持状态码切换。
- 支持动态 Mock 数据。
- 支持 Mock 请求日志。
- 支持按路径和方法匹配 Mock 规则。

### 12. API 文档

- 支持从请求生成接口文档。
- 支持参数说明、Header 说明、Body 示例、响应示例。
- 支持 Markdown / HTML 导出。
- 支持文档与请求集合联动。
- 支持接口描述、标签和备注。

### 13. 导入导出

- 导入 Postman Collection。
- 导入 OpenAPI / Swagger。
- 导入 cURL。
- 导出 cURL。
- 导出 Holaman 项目文件。
- 导出 Collection。
- 导出环境变量。

### 14. WebSocket / SSE

- 支持 WebSocket 连接调试。
- 支持消息发送与接收。
- 支持消息历史。
- 支持连接断开和重连。
- 支持自定义 WebSocket Header。
- 支持 SSE 流式响应查看。

### 15. GraphQL

- 支持 GraphQL Query / Mutation 编辑。
- 支持 Variables 编辑。
- 支持 Schema introspection。
- 支持字段自动补全。
- 支持 GraphQL 响应格式化。
- 支持 GraphQL 请求保存到 Collection。

### 16. 版本管理

- 支持请求修改历史。
- 支持 Collection 版本。
- 支持接口变更 diff。
- 支持回滚历史版本。
- 支持版本备注。

### 17. 编辑器体验

- 支持 JSON 自动格式化。
- 支持语法高亮。
- 支持 Header 名称自动补全。
- 支持变量自动提示。
- 支持错误 JSON 提示。
- 支持快捷键发送请求。
- 支持多标签页并行调试。

### 18. 搜索与导航

- 支持全局搜索请求。
- 支持按 URL、方法、名称、Header 搜索。
- 支持快捷键。
- 支持标签页切换。
- 支持快速打开 Collection、文件夹和请求。

### 19. 安全与隐私

- 支持敏感变量加密存储。
- 支持 Token 脱敏展示。
- 支持本地优先存储。
- 支持导出时排除敏感字段。
- 支持请求日志敏感信息遮罩。

## 暂不纳入第一阶段

以下能力暂时不进入第一阶段，后续根据产品发展再规划：

- 团队协作。
- 插件系统。
- CI / 命令行自动化集成。
